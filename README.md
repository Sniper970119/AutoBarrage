# Sniper

使用Selenium 自动发送斗鱼弹幕

*免责声明：仅供研究使用，请勿进行非法用途。*

## 环境要求

1.python 3

2.Selenium

#
使用方法：

双击<code>douyudanmu.exe</code>

在<code>danmu.dm</code>中配置发送的弹幕，在<code>config.py</code>中配置房间号以及发送间隔。

第一次使用需要登录，会自动保存cookies，所以一个帐号只能使用一个文件，多账号需要多文件。
